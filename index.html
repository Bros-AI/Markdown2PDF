üõ†Ô∏è Installation & Setup

Option 1: Direct Usage

Simply visit mark2pdf.com to use the tool immediately - no installation required!

Option 2: Self-Hosting

  1. Clone the repository:

    git clone https://github.com/BrosG/markdown-to-pdf.git

  2. Open the index.html file in your browser or serve it with a simple HTTP server:

    # Using Python
python -m http.server

# Using Node.js
npx serve

  3. Access the application at http://localhost:8000 (or the port provided by your server)

üìñ Usage Guide

Basic Usage

  1. Type or paste your Markdown in the editor panel (left side)
  2. View the rendered output in the preview panel (right side)
  3. Click "Generate PDF" to download your document as a PDF

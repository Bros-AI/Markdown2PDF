Keyboard Shortcuts

Use these keyboard shortcuts to speed up your workflow:

  • Ctrl + B - Bold
  • Ctrl + I - Italic
  • Ctrl + K - Insert Link
  • Ctrl + H - Toggle Heading
  • Ctrl + U - Unordered List
  • Ctrl + O - Ordered List
  • Ctrl + Q - Quote
  • Ctrl + Shift + C - Code Block
  • Ctrl + S - Save Markdown
  • Ctrl + P - Generate PDF
  • Ctrl + Z - Undo
  • Ctrl + Y - Redo
